# 🏸 Badminton Tournament Organizer

A Python + MySQL project to manage badminton tournaments, quick matches, player stats, and game organization.

## 🚀 Features
- User Signup / Signin
- Start Tournament
- Continue Tournament
- Quick Match Mode
- Quick Game Mode
- Match Statistics
- MySQL Database Integration

## 🛠 Tech Stack
- Python
- MySQL
- VS Code

## 📂 Project Structure
See folders:
- auth/
- database/
- modes/
- tournament/
